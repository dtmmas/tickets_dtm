COMANDOS CLAVE GIT

Ver estado:
git status

Agregar cambios:
git add .

Crear commit:
git commit -m "Describe aqui el cambio"

Subir cambios:
git push origin main

Bajar cambios en VPS:
git pull origin main


COMANDOS CLAVE VPS

Entrar por SSH:
ssh root@TU_IP

Ir al proyecto:
cd /var/www/tickets_dtm

Ir al backend:
cd /var/www/tickets_dtm/server

Instalar frontend:
npm install

Generar build:
npm run build

Instalar backend:
cd /var/www/tickets_dtm/server
npm install

Reiniciar PM2:
pm2 restart tickets-dtm --update-env

Ver logs:
pm2 logs tickets-dtm --lines 20

Ver estado PM2:
pm2 status

Vaciar logs PM2:
pm2 flush


COMANDOS CLAVE NGINX

Editar configuracion:
nano /etc/nginx/sites-available/tickets-dtm

Validar configuracion:
nginx -t

Reiniciar Nginx:
systemctl restart nginx

Ver estado:
systemctl status nginx


COMANDOS CLAVE MYSQL

Ver estado:
systemctl status mysql

Entrar a MySQL:
mysql -u root -p

Ver tablas:
USE ticket_system;
SHOW TABLES;
