COMANDOS VPS - TICKETS DTM

1. PLANTILLA CORTA DE 10 COMANDOS

ssh root@TU_IP
cd /var/www/tickets_dtm
git pull origin main
npm install
npm run build
cd /var/www/tickets_dtm/server
npm install
pm2 restart tickets-dtm --update-env
pm2 logs tickets-dtm --lines 20
exit


2. MANUAL CORTO

Entrar al VPS:
ssh root@TU_IP

Ir al proyecto:
cd /var/www/tickets_dtm

Descargar cambios:
git pull origin main

Instalar dependencias frontend:
npm install

Generar build frontend:
npm run build

Ir al backend:
cd /var/www/tickets_dtm/server

Instalar dependencias backend:
npm install

Si hubo cambios en .env:
nano /var/www/tickets_dtm/server/.env

Reiniciar aplicacion:
pm2 restart tickets-dtm --update-env

Ver logs:
pm2 logs tickets-dtm --lines 20

Ver estado:
pm2 status

Si hubo cambios en Nginx:
nano /etc/nginx/sites-available/tickets-dtm
nginx -t
systemctl restart nginx

Verificaciones rapidas:
systemctl status nginx
systemctl status mysql
curl http://127.0.0.1:3001/api/config
curl https://app.sistemasdtm.com/api/config
